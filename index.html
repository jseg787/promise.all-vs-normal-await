This is a comparison between using multiple awaits in an async function vs using Promise.all()

Each line using 'await' pauses and waits for the promise to resolve before continuing, causing the final result to be very slow

Using Promise.all() causes all calls to happen at the same time so there is no hold up in the code execution

The asynchronous requests are simulated using setTimeout() with a resolved promise

Each side is making 10 requests
